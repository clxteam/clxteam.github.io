📱 Add Developer Shortcut

Tạo iOS Shortcut để thêm developer từ App Store Share Sheet

🔧 Thiết lập Shortcut

1

Tạo Shortcut mới

Mở app Shortcuts → bấm + → đặt tên "Add Developer"

Bấm icon ⓘ ở trên cùng → bật Show in Share Sheet
Ở dòng "Receives": chọn URLs, Text, và Safari web pages

2

Action: Get URLs from Input

Tìm action Get URLs from Input
Input: chọn Shortcut Input

⚠️ Bước này rất quan trọng — khi share từ App Store, iOS gửi text dạng:
"App Name by Developer https://apps.apple.com/..."
Action này sẽ trích xuất URL ra khỏi đoạn text đó.

3

Action: Get Contents of URL (POST request)

Tìm action Get Contents of URL

Cấu hình:
- URL: bấm vào ô URL → nhập URL API bên dưới


- Bấm Show More
- Method: POST
- Headers: thêm 1 header → Key: Content-Type Value: application/json
- Request Body: chọn JSON
- Thêm 1 field → Key: url → Value: bấm chọn URLs (kết quả từ bước 2)

4

Action: Get Dictionary Value

Tìm action Get Dictionary Value
- Get: Value
- for Key: nhập developerName
- in: chọn Contents of URL (kết quả từ bước 3)

5

Action: Show Notification

Tìm action Show Notification
Nội dung: nhập "Đã thêm " rồi chọn Dictionary Value (kết quả từ bước 4)

💡 Cách dùng: Vào App Store → giữ vào app → Share → kéo xuống chọn "Add Developer".

Nếu không thấy shortcut trong Share Sheet: kéo xuống cuối → bấm Edit Actions → bật shortcut lên.
⚠️ Lưu ý: Nếu shortcut chạy rất nhanh mà không có notification → kiểm tra lại bước 2 (Get URLs from Input). Thiếu bước này sẽ khiến URL rỗng và API trả lỗi ngay lập tức.
