📱 Add Developer Shortcut

Hướng dẫn tạo iOS Shortcut để thêm developer từ App Store

1

Mở app Shortcuts → tạo Shortcut mới

Bật "Show in Share Sheet" và chọn nhận "URLs" và "Text"

2

Thêm action: URL

Nhập URL sau:

3

Thêm action: Get Contents of URL

- Method: POST
- Headers: Content-Type = application/json
- Request Body: JSON
- Thêm key url với value là Shortcut Input

4

Thêm action: Get Dictionary Value

Key: developerName từ "Contents of URL"

5

Thêm action: Show Notification

Nội dung: Đã thêm [Dictionary Value]

💡 Cách dùng: Vào App Store → giữ app → Share → chọn shortcut "Add Developer". Chỉ cần 5 action, không cần logic phức tạp vì server xử lý hết.
